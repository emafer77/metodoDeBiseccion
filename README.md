## Requisitos

- Python 3.10.12
- Dependencias listadas en `requirements.txt`

## Instalación

1. **Clona el repositorio** o descarga el archivo comprimido del proyecto:

   ```bash
   git clone https://github.com/emafer77/metodoDeBiseccion.git
   cd tu_proyecto
   ```

2. **Crea y activa un entorno virtual** (opcional, pero recomendado):

   En macOS y Linux:

   ```bash
   python3 -m venv env
   source env/bin/activate
   ```

   En Windows:

   ```bash
   python -m venv env
   .\env\Scripts\activate
   ```

3. **Instala las dependencias** usando el archivo `requirements.txt`:

   ```bash
   pip install -r requirements.txt
   ```

## Ejecución del programa

Para ejecutar el programa, utiliza el siguiente comando:

- en linux o mac

```bash
python3 examen1.py
```

- en Windows

```bash
python examen1.py
```

- se creara una imagen `mi_grafica.png`despues de correr el programa con la grafica de muestra.
